## 개요

지구의 블랙홀화 시뮬레이터입니다. 달과 행성부터 별, 은하까지 차례로 삼키며 점점 커집니다.

## 플레이 방법

<div align="center">

우측 상단 **행복 버튼(☺)** 을 누르면 시작합니다.
10초 뒤 지구가 블랙홀로 변하기 시작하고, 30초에 걸쳐 완전한 블랙홀이 됩니다.

| 키 | 동작 |
| :---: | :---: |
| `W` `A` `S` `D` | 이동 |
| `Space` / `Shift` | 상승 / 하강 |
| 마우스 드래그 · 휠 | 시점 회전 · 줌 |
| 천체 이름 클릭 | 그곳까지 자동 비행 |
| `J` `H` `X` | 기록 · 중력 강화 · 좌표축 |

![블랙홀이 되는 과정](assets/collapse.gif)

</div>

## 특징

- 데이터는 저장되지 않습니다.
- 태양계부터 이웃 은하까지 실제 거리 비율로 구현
